UniGetUI (icon database)
https://github.com/Devolutions/UniGetUI

Copyright (c) 2025 Marti Climent

Winhance fetches the UniGetUI community icon database
(WebBasedData/screenshot-database-v2.json) at runtime to resolve icons for
WinGet packages whose own manifests don't provide an Icons URL. Winhance does
not bundle, modify, or redistribute the database file or any icon image —
each app launch reads the latest copy directly from the upstream repository.
