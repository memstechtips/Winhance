# Icon attribution & licenses

Single source of truth for the provenance and licensing of the image assets under
`Assets/`. Entries are organized by **provenance** (where the art came from), not by
folder. When you add a new icon — in `ModeIcons/`, a future `FeatureIcons/`, anywhere —
add an entry here. Do **not** create per-folder `LICENSE.txt` files.

These asset licenses are independent of the Winhance source-code license (PolyForm
Shield 1.0.0, see the repository-root `LICENSE.txt`).
